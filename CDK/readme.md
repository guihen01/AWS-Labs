# AWS CDK Learning Series

Hands-on AWS Cloud Development Kit (CDK) learning series focused on Infrastructure as Code (IaC), cloud automation, serverless architectures, networking, and DevOps practices using Python.

This repository series documents progressive AWS CDK projects ranging from beginner foundations to more advanced cloud-native infrastructure deployments.

---

# Objectives

The goal of this learning series is to:

- Build practical AWS CDK experience
- Explore Infrastructure as Code using Python
- Compare Terraform and AWS CDK approaches
- Develop reusable cloud deployment patterns
- Practice modern AWS cloud architectures
- Improve DevOps and cloud engineering skills
- Build a professional cloud portfolio on GitHub
